# mains-leisu-notes

一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为个人笔记和页面的存放与展示空间。

## 项目简介

`mains-leisu-notes` 旨在收集、整理和发布各类独立的 HTML 页面。这些页面可能是学习笔记、工具页面、实验性项目或临时文档。每个页面均为自包含的 HTML 文件，便于直接查看和分享。

## 目录说明

```
.
├── index.html          # 仓库首页或导航页（可选）
├── pages/              # 存放所有独立 HTML 页面
│   ├── example-page.html
│   └── ...
├── assets/             # 公共资源（如 CSS、JS、图片等）
│   ├── styles/
│   ├── scripts/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：所有独立 HTML 页面均放置于此目录下，按需要可进一步分子目录。
- `assets/`：存放页面共用的样式、脚本和图片等资源文件。
- `index.html`：可选的导航页，用于快速浏览仓库中的页面列表。

## 页面归档说明

- 每个 HTML 页面应尽量保持独立，避免依赖外部资源（除非通过 `assets/` 目录引用）。
- 页面文件名应具有描述性，便于识别内容。
- 如需引用仓库内资源，请使用相对路径，例如 `../assets/styles/main.css`。
- 新增页面时，建议在 `pages/` 下创建对应的文件夹（可选）或直接放置文件。
